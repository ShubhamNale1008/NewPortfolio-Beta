## About

I'm **Shubham Rajendra Nale**, a Computer Science & Technology student at **Shivaji University, Kolhapur**. This portfolio is a premium, fully responsive single-page application showcasing my skills, projects, certifications, and journey as a developer focused on AI and web development.

---

## Features

| Feature | Description |
|---|---|
| Animated Loader | Custom branded intro animation with ring progress |
| Smooth Scroll Animations | Reveal-on-scroll effects throughout all sections |
| PDF.js Resume Viewer | In-browser resume viewer with download option |
| Certificate Gallery | Lightbox modal for viewing full-size certificates |
| Contact Form | Functional form integrated with Web3Forms API |
| Keyboard Shortcuts | Navigate sections with `H`, `A`, `E`, `S`, `P`, `W` keys |
| Fully Responsive | Optimized for mobile, tablet, and desktop |
| SEO & Meta Tags | OpenGraph, Twitter Card, canonical URL, robots meta |
| CSP Headers | Content-Security-Policy for secure deployment |
| Fast Deployment | Hosted on Vercel with zero-config CI/CD |
